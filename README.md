The data will be deleated as the user disconnect from the Server.

# Backend 

- intialize a Node.js project 
    npm init -y

- add typescript and ws 
    npm install typescript 
    npx tsc --init 
    npm i ws @types/ws

# Frontend
- npm create vite@latest
    framework: React
    variant: TypeScript
- npm install tailwindcss @tailwindcss/vite
